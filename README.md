# 🎬 Cinzey OTT

A modern OTT streaming platform built with React.js and Vite, offering a responsive and engaging user experience inspired by popular streaming services.

## 🚀 Features

* Responsive and modern UI
* Dynamic movie and TV show listings
* Reusable React components
* Fast performance with Vite
* Mobile-friendly design
* Clean and scalable project structure
* API integration for fetching content

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* Vite
* HTML5
* CSS3

## 📂 Project Structure

src/
├── assets/
├── components/
├── pages/
├── api/
├── App.jsx
└── main.jsx

## ⚙️ Installation

1. Clone the repository

git clone https://github.com/aishwaryapratapsigh/cinzey-ott.git

2. Navigate to the project folder

cd cinzey-ott

3. Install dependencies

npm install

4. Start development server

npm run dev
